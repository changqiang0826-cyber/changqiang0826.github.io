📝 使用说明（部署前请删除这整段）

1. 设置密码：在文件底部的 <script> 中找到 const PASSWORD = "520"，改成你想要的密码。

2. 写信：修改上面 class="body-text" 里的文字，直接写你想说的话。

3. 换照片：把 <img src="..."> 里的地址换成你的照片。可以用图床（如 imgbb.com 免费上传），或者把照片文件和 HTML 放在同一个文件夹里，直接写文件名如 src="photo1.jpg"。

4. 加照片：复制一个 photo-card 的 div 就能增加新照片。

5. 照片说明：修改每张照片下方 class="caption" 里的文字。

6. 署名和日期：修改 class="signature" 和 class="date" 里的内容。

7. 部署推荐：

　　• 最简单：用 GitHub Pages（免费），把文件上传到仓库即可得到链接。

　　• 更简单：用 Netlify Drop（免费），直接拖拽文件夹上传，秒得链接。

　　• 最快：用 Vercel（免费），连 GitHub 仓库自动部署。

⚠️ 安全提示：这个密码保护是前端级别的，适合防止普通人直接看到内容。懂技术的人可以通过查看源代码绕过。对于写情书来说完全够用了！
